Bloque 1: Estudio de Mercado (Smart TV)

Funciones iniciales: Demanda: q = 5.000 - 10p | Oferta: q = 5p

a) Precio y cantidad de equilibrio

Igualamos cantidad ofertada y demandada (qo = qd):

5.000 - 10p = 5p ⇒ 5.000 = 15p ⇒ p* = 333,33

Sustituyendo en la oferta: q* = 5 · 333,33 = 1.666,67 unidades

b) Excedente del Consumidor (EC)

Precio máximo (donde q=0): pmax = 500

EC = [(500 - 333,33) · 1.666,67] / 2

EC = 138.889,17

c) Excedente del Productor (EP)

Precio mínimo (donde q=0): pmin = 0

EP = [(333,33 - 0) · 1.666,67] / 2

EP = 277.778,33

Escenario con Impuesto (La cantidad cae a qi = 500)

Precios bajo el efecto del impuesto: pd (consumidor) = 450 | po (productor) = 100

  • d) Excedente del consumidor con impuesto:
    ECi = [(500 - 450) · 500] / 2 = 12.500
  • e) Excedente del productor con impuesto:
    EPi = [(100 - 0) · 500] / 2 = 25.000
  • f) Recaudación Fiscal:
    Impuesto unitario (T) = 450 - 100 = 350. Recaudación = 350 · 500 = 175.000
  • g) Pérdida de excedente del consumidor:
    138.889,17 - 12.500 = 126.389,17
  • h) Pérdida de excedente del productor:
    277.778,33 - 25.000 = 252.778,33
  • i) Pérdida de excedente total para la sociedad (PSN):
    PSN = [ (450 - 100) · (1.666,67 - 500) ] / 2 = 204.167,50

Bloque 2: Gerencia en Mercado Competitivo (Relojes)

Datos: CT = 200 + 2q2 | CMg = 4q | CF = 200 | Precio (P) = 100

a) Nivel de producción óptimo

Regla de maximización en competencia perfecta: P = CMg

100 = 4q ⇒ q = 100 / 4 ⇒ q = 25 unidades

b) Nivel de utilidad

  • Ingreso Total (IT) = 100 · 25 = 2.500
  • Costo Total (CT) = 200 + 2(25)2 = 1.450
  • Utilidad (B) = 2.500 - 1.450 = 1.050 dólares

Bloque 3: Ejercicios Variados de Competencia Perfecta

Pregunta 1: Firma competitiva (P = 375)

Funciones: CT = 1000 + 2,5q2 | CMg = 5q

  • a) Producción óptima: 375 = 5q ⇒ q = 75
  • b) Ingreso Total: 375 · 75 = 28.125
  • c) Costo Total: 1000 + 2,5(75)2 = 15.062,5
  • d) Utilidad: 28.125 - 15.062,5 = 13.062,5
