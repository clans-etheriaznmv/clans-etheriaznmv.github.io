¿Qué trae de nuevo esta actualización?

¡Saludos, exploradores! Les presentamos el nuevo sistema de clanes, que a pesar de iniciar sólo como una idea en desarrollo, esperamos que pueda utilizarse para realizar eventos y dinámicas divertidas junto a todos ustedes. Esta actualización trae consigo la introducción de un sistema formal para la creación de grupos (llámese clanes) que podrán competir cada cierto tiempo para obtener recompensas exclusivas tanto dentro del Servidor, como cosméticos para la misma aplicación. Puedes encontrar los cambios más relevantes y destacables a continuación:

Cosas Nuevas

● [26-04-2026] Implementación del rol "Líder de Clan".
● [26-04-2026] Implementación del rol "Co Líder de Clan".
● [26-04-2026] Implementación del rol "Miembro de Clan".
● [26-04-2026] Creación de un canal exclusivo junto a la creación del clan.
● [26-04-2026] Cada clan tendrá un rol de identificador con color e ícono personalizado.
● [26-04-2026] El Líder del Clan obtendrá 4 remociones de advertencias para todo su clan.
● [26-04-2026] Se fijó la cantidad de miembros por clan a 5.

Cambios Pendientes

● [PENDIENTE] Definir día y recompensas para la primera guerra de clanes.
● [PENDIENTE] Sistema de puntos de clan para comprar experiencia dentro del servidor y cosméticos para la aplicación.
● [PENDIENTE] Tienda personalizada para usar los puntos de clan.
● [PENDIENTE] Sorteos exclusivos para los miembros de un clan.

Cambios Internos

● [27-04-2026] A la hora de mejorar el Servidor, se deberá elegir entre las recompensas o la fundación de un clan; ambas no son acumulables.
● [27-04-2026] Las personas que ya hayan reclamado sus recompensas por mejorar el servidor, podrán fundar un clan.
● [27-04-2026] En caso de tener un rol personalizado y fundar un clan, el mismo será eliminado.
